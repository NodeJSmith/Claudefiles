- `skills/mine.plan/SKILL.md` produces `tasks/context.md` (master context file) and `tasks/T{NN}-{slug}.md` task files instead of `tasks/WP*.md` work package files

- The master context file contains all five required sections: Problem & Motivation, Visual Artifacts, Key Decisions, Constraints & Anti-Patterns, Design Doc References

- The reviewer-prompt.md is rewritten to validate traceability and prompt quality instead of the current 10-point WP checklist

- FR/AC identifier format is validated (FR#13a)
