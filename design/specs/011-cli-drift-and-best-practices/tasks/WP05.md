4. For `bin/gh-issue`: add `--help`/`-h` guard before the bot-token preamble. Show examples wrapping common `gh issue` subcommands.
5. For `bin/gh-pr-create`: add `--help`/`-h` guard before the bot-token preamble. Show examples.
6. For `bin/claude-log`: add `epilog` with examples to the main parser and key subparsers (list, show, search, grep, stats). Remove `from __future__ import annotations` (line 17) and fix any type annotations that relied on it.
7. For `bin/claude-merge-settings`: move `import argparse` from inside `main()` to top of file. Add `epilog` with examples to the parser.
8. For `bin/agnix-check`: verify `--help` works (it's a Python script â€” check if argparse or a manual handler exists). Add one if missing, with examples.
