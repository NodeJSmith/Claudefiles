## Objectives & Success Criteria

- A `Turn` TypedDict exists that converts raw JSONL entry dicts into a domain object with `role`, `text`, `tool_calls`, `timestamp`, `request_id`, and `source` fields
- A `to_turn()` function converts entries; entries with the same `requestId` are collapsed into a single Turn
- An `_output()` dispatch function routes to JSON (`json.dumps`) or text (formatter callable) based on `args.json`
- `--json` flag is added to each subcommand parser individually (works in any position)
- `claude-log list` produces a columnar text table by default, JSON with `--json`
- `claude-log stats <id>` produces a key-value text summary by default, JSON with `--json`
- `from __future__ import annotations` is removed from `tests/test_claude_log.py`
- All existing tests pass; new tests cover Turn model, `_output` dispatch, and text formatters

## Subtasks

1. Remove `from __future__ import annotations` from `tests/test_claude_log.py:2`
2. Define `Turn` TypedDict in `bin/claude-log` (after the existing helper section, before command handlers): `role`, `text`, `tool_calls`, `timestamp`, `request_id`, `source`
3. Implement `to_turn(entry: dict, agent_index: dict) -> Turn` that uses `extract_text()` and `_tool_input_summary()` to populate fields
4. Implement `to_turns(entries: list[dict], agent_index: dict) -> list[Turn]` that groups entries by `requestId` and collapses multi-entry assistant responses into single Turns (thinking blocks included but flagged)
5. Implement `_output(data, *, args, formatter)` dispatch function
6. Add `--json` flag to each subcommand parser in `build_parser()` (not the global parser)
7. Create `_format_list(sessions: list[dict]) -> str` â€” columnar table with date, project, branch, messages, duration
8. Create `_format_stats(stats: dict) -> str` â€” key-value summary
9. Update `cmd_list` to call `_output(sessions, args=args, formatter=_format_list)` instead of `print(json.dumps(...))`
10. Update `cmd_stats` to call `_output(stats, args=args, formatter=_format_stats)` instead of `print(json.dumps(...))`
