# Interaction Style

## Clarify, Don't Plan

When a task is ambiguous or has multiple valid approaches, do NOT use the
`EnterPlanMode` tool. Instead, use `AskUserQuestion` to clarify the specific
points where the correct choice is unclear. Ask focused, minimal questions —
only what's needed to proceed confidently. Then start implementing immediately
after getting answers.

Only use `EnterPlanMode` if the user explicitly requests it (e.g., "enter plan
mode", Shift+Tab toggle).
